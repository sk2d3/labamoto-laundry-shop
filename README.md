# labamoto laundry shop

 A Windows Forms–based Laundry Service Management System designed to replace manual, paper-based laundry operations. The system digitizes customer records, automates service fee calculations, tracks laundry status in real time, and generates QR code–enabled digital receipts for easy order monitoring. It also provides transaction history, income summaries, and incident reporting tools to improve operational efficiency, transparency, and accountability for a local laundry shop.
